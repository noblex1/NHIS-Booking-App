# 🔐 Password Management - Quick Guide

## 🎯 Two Ways to Manage Your Password

### 1. 🔒 Change Password (When Logged In)
**Use this when**: You know your current password and want to change it

**Location**: Profile → Change Password

**Steps**:
```
1. Login to your account
2. Go to Profile page
3. Click "Change Password"
4. Click "Send Verification Code"
5. Check your email for OTP
6. Enter OTP + New Password
7. Done! ✅
```

---

### 2. 🔑 Forgot Password (When Logged Out)
**Use this when**: You forgot your password and can't login

**Location**: Login → Forgot Password? Reset

**Steps**:
```
1. Go to Login page
2. Click "Forgot Password? Reset"
3. Enter your email address
4. Click "Send Reset Code"
5. Check your email for OTP
6. Enter OTP + New Password
7. Login with new password ✅
```

---

## 📧 OTP Email

Both methods send you a **6-digit verification code** via email:

```
Subject: Your NHIS Verification Code

Your verification code is: 123456

This code will expire in 10 minutes.

If you didn't request this code, please ignore this email.
```

## 🔒 Security Features

| Feature | Description |
|---------|-------------|
| **OTP Expiration** | Code expires in 10 minutes |
| **Max Attempts** | 5 attempts per code |
| **Rate Limiting** | Prevents spam requests |
| **Password Hashing** | Secure bcrypt encryption |
| **Email Verification** | Confirms your identity |

## ✅ Password Requirements

- ✅ Minimum 6 characters
- ✅ Must match confirmation
- ✅ Cannot be empty

---

## 🚨 Common Errors & Solutions

### "Invalid or expired OTP"
**Solution**: Click "Resend Code" to get a new OTP

### "Passwords do not match"
**Solution**: Make sure both password fields are identical

### "Password must be at least 6 characters"
**Solution**: Use a longer password (minimum 6 characters)

### "Too many requests"
**Solution**: Wait a few minutes before trying again

### "OTP not received"
**Solutions**:
- Check spam/junk folder
- Verify email address is correct
- Wait 1-2 minutes for delivery
- Click "Resend Code"

```

```

## 💡 Tips

1. **Use a strong password**: Mix letters, numbers, and symbols
2. **Don't share your password**: Keep it private
3. **Change regularly**: Update your password periodically
4. **Save OTP emails**: Keep them until you complete the process
5. **Act quickly**: OTP expires in 10 minutes
6. **Check spam**: OTP emails might go to spam folder
