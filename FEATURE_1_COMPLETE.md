## 🎯 How to Use

### Step 1: Access User Details
```
Admin Dashboard → Users → Click 👁️ icon
```

### Step 2: View Information
- See complete user profile
- Check appointments
- Review verification status

### Step 3: Edit (Optional)
```
Click [Edit] → Make changes → Click [Save Changes]
```

### Step 4: Delete (Optional)
```
Click [Delete] → Confirm → User deleted
```
