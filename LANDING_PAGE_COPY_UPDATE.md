Updated the landing page title and description to be more engaging and removed the passport application reference.

## Why These Changes?

### ✅ More User-Friendly
- "Skip the queues" - Immediately communicates the main benefit
- "with ease" - Emphasizes simplicity
- "walk in with confidence" - Creates emotional connection

### ✅ Removed Technical Jargon
- ❌ Removed "passport application booking" comparison
- ❌ Removed technical terms like "OTP verification"
- ✅ Used simple, clear language

### ✅ Benefit-Focused
- Focuses on what users gain (no queues, confidence, ease)
- Rather than just describing features
- More persuasive and engaging

### ✅ Emotional Appeal
- "with confidence" - Addresses user anxiety
- "hassle-free" - Promises smooth experience
- "secure your spot" - Creates sense of control
